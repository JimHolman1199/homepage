GitHub Pages: 
    * prod: https://jimholman1199.github.io/homepage/

# Start with **git**

1. `git clone https://github.com/JimHolman1199/homepage.git`

2. `cd homepage`

3. `serve`

### Verify the installation

* Open in browser: `localhost:5000` or `http://192.168.0.15:5000 `
